# 🛡️ JobShield Pakistan - AI Scam Detector

**JobShield Pakistan** is a web-based tool designed to protect job seekers in Pakistan from online recruitment fraud. Using Google Gemini AI, the system analyzes job descriptions, WhatsApp messages, and advertisements to identify common scam indicators.

## 🚀 Live Demo
Check out the live application here: [https://job-shield-pakistan.vercel.app](https://job-shield-pakistan.vercel.app)

## ✨ Key Features
- **AI-Powered Analysis:** Leverages Google Gemini (Flash) for high-accuracy scam detection.
- **Language Support:** Analyzes content in both **English** and **Roman Urdu**.
- **Risk Scoring:** Provides a risk score (0-100) based on identified red flags.
- **Detailed Red Flags:** Points out specific issues like "Pay-to-work" schemes, generic emails, and unprofessional language.
- **Premium UI:** Modern dark-themed design with a focus on usability and accessibility.
- **Secure Backend:** Uses Vercel Serverless Functions to securely handle AI requests.

## 🛠️ Built With
- **Frontend:** HTML5, Vanilla CSS, JavaScript (ES6+)
- **Backend:** Node.js (Vercel Serverless Functions)
- **AI:** Google Gemini API
- **Database:** Firebase Firestore (for reported scams)

## 🛡️ Privacy & Safety
JobShield does not store personal information from analyzed texts. It is built as a community tool to help Pakistanis stay safe in the digital job market.
